# unbothered.ai

A modern React-based journaling website with Sign Up, Sign In, and Journaling pages. Built with Vite.

## Pages
- **Sign Up**: Create a new account
- **Sign In**: Log in to your account
- **Journaling (Unbothered)**: Write and view your journal entries

## Getting Started

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Start the development server:
   ```powershell
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack
- React
- Vite
- react-router-dom
